# Restful-API-For-Inventory
Restful API using Laravel

A Laravel-based RESTful API for inventory management with full CRUD operations.

## Features  
- **Resource Endpoints**: `GET/POST/PUT/DELETE /api/items`  
- **Validation**: Request data sanitization  
- **Database**: Eloquent ORM with migrations  
- **Testing**: Postman-ready endpoints  

## Technologies  
- **Backend**: Laravel 9  
- **Database**: MySQL  
- **Tools**: Postman for testing
